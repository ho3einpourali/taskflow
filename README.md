# TaskFlow

TaskFlow is a simple Kanban board application built using HTML, CSS, and JavaScript.

It helps users organize tasks visually with drag-and-drop functionality, task management, and dark/light theme support.

## Features

- Drag and drop tasks between columns
- Add, edit, and delete tasks
- Dark and light mode toggle
- Local storage persistence
- Responsive layout
- Clean modern UI
